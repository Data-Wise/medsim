# Format Number in Scientific Notation for LaTeX

Format Number in Scientific Notation for LaTeX

## Usage

``` r
.format_scientific_latex(x, digits = 1, threshold = 0.001)
```

## Arguments

- x:

  Numeric value

- digits:

  Number of significant digits in mantissa

- threshold:

  Use scientific notation if abs(x) \< threshold

## Value

Character string with LaTeX scientific notation
