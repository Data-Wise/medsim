# Format Percentage for LaTeX

Format Percentage for LaTeX

## Usage

``` r
.format_percentage_latex(x, digits = 1, as_proportion = TRUE)
```

## Arguments

- x:

  Numeric value (proportion 0-1 or percentage 0-100)

- digits:

  Decimal places

- as_proportion:

  If TRUE, x is 0-1; if FALSE, x is 0-100

## Value

Character string with percentage
