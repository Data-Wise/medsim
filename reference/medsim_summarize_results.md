# Summarize Simulation Results

Computes summary statistics from simulation results including mean,
median, standard deviation, and quantiles.

## Usage

``` r
medsim_summarize_results(results, by = "scenario")
```

## Arguments

- results:

  data.frame of simulation results

- by:

  Character vector: columns to group by (default: "scenario")

## Value

data.frame with summary statistics
